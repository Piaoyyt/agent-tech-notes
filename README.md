# Agent技术笔记

> 大模型算法工程师的 Agent 技术沉淀与实战记录。

目前在大模型算法领域从事相关工作，日常核心方向是 **Agent 系统设计与落地**——把传统工作流演进为 Agent-to-Agent（A2A）智能协作架构。

这个仓库是公众号 **「Agent技术笔记」** 的配套代码和文章归档。
